jQuery概念

它是一个快速和简洁的JavaScript库，为了简化javascript的开发。
它的理念是：写得少，做得更多的轻量级JavaScript库。
它的优势：轻量级，强大的选择器(XXX选择器)，出色的事件处理机制，完善的Ajax，出色的浏览器兼容，
，链式编程操作,还有丰富的插件。

HTML/XHTML区别

    HTML是一种基于web的网页设计语言,XHTML是一个基于XML的置标语言,XHTML严谨性比HTML高,然后
基本语言都是沿用HTML标签.
XHTML相对HTML的几大区别
XHTML要求正确的嵌套、所有的元素必须关闭、区分大小写、属性值要用双引号、用id属性代替name属性

svn、git都含有以下过程：

1、搭建一个git的服务器，服务器就有相应的地址、管理员
2、开发者A：

2.1、管理员给A分配一个账户
2.2、将账户和服务器进行关联
2.3、编写代码
2.4、直接提交到服务器

3、开发者B(最常用的角色)：
3.1、管理员给B分配一个账户
3.2、将账户和服务器进行关联(下载服务器代码到本地)
3.3、编写代码
3.4、更新服务器代码
3.4.1、更新完毕没有冲突，稍微检查一下功能有没有错误
3.4.2、更新的过程发生了冲突，需要解决一下冲突，解决完冲突，重新测试下自己的功能
3.5、提交代码
4、后续的任何开发者的操作都应该重复3.3-3.5这个过程

Git的基本使用

git status：查看仓库中的文件状态

git add：将文件放到暂存区

关于用户名和邮箱的配置

git push：将本地主仓库中的代码上传到服务器中

git pull：将服务器代码更新到本地(关联)

git clone：将服务器代码下载到本地(将本地仓库跟服务器仓库进行关联)
